# TripEasy






## Instruções:

Para executar o projeto:

* clone este repositório com `git clone https://github.com/Taina-conde/tripeasy.git`
* `cd tripeasy`
* instale todas as dependências com `npm install`
* inicie o ambiente de desenvolvimento com `npm start`

## Tecnologias Usadas

* JavaScript
* React
* Bootstrap
* Sass
* Formik
